# The-Seed-Of-Hope
The Seed of Hope Dashboard is an interactive platform designed to analyze and combat hidden hunger in Rwanda using data from the Demographic and Health Survey (DHS). It visualizes key nutrition indicators such as stunting, wasting, and underweight prevalence. The dashboard features data driven insights and proposes solutions like IoT-based greenhouses, nutrition education, maternal and child health programs, free school meals, and food fortification. It empowers policymakers and communities with evidence-based strategies to promote food security and improve national nutrition outcomes sustainably
